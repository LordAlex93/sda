Sitio web educativo desarrollado con **Quarto** para la creación y publicación de materiales interactivos de **matemáticas** y **criptografía**.

---

## Descripción

**SDA** es un proyecto web orientado a la elaboración de recursos didácticos en formato digital, combinando explicación teórica, actividades contextualizadas e interactividad matemática.

El sitio reúne materiales sobre:

- **coordenadas cartesianas** y representación en el plano,
- **formas de representar funciones**,
- **funciones lineales, afines y constantes**,
- **funciones cuadráticas**,
- y una **misión final de criptografía** basada en cifrado César y RSA.

La web está construida con **Quarto**, lo que permite trabajar con una estructura clara de documentos `.qmd`, generar páginas HTML de forma reproducible y mantener un flujo de trabajo limpio con Git y GitHub.

---

## Objetivos del proyecto

- Diseñar y publicar materiales didácticos en formato web.
- Integrar interactividad matemática en el proceso de aprendizaje.
- Organizar contenidos de manera modular y reutilizable.
- Facilitar la edición, el mantenimiento y la evolución del sitio.
- Unificar recursos de matemáticas y criptografía dentro de una misma plataforma.

---

## Características principales

- **Arquitectura basada en Quarto** para la generación de sitios web estáticos.
- **Actividades interactivas con GeoGebra** integradas directamente en las páginas.
- **Diseño modular por sesiones y bloques temáticos**.
- **Enfoque didáctico y contextualizado**, con narrativa de misiones y resolución guiada.
- **Preparado para publicación en GitHub Pages**.

---

## Contenidos destacados

### Coordenadas cartesianas
Actividades de localización de puntos, clasificación por cuadrantes, desplazamientos y rutas sobre el plano cartesiano.

### Representación de funciones
Propuestas para trabajar la representación **verbal**, **tabular**, **gráfica** y **simbólica** de funciones.

### Funciones lineales
Material sobre funciones lineales, afines y constantes, incluyendo representación gráfica, interpretación de pendiente y valor inicial.

### Funciones cuadráticas
Estudio de parábolas, tablas de valores, sentido de apertura y análisis de máximos y mínimos.

### Criptografía
Actividad final en grupo
