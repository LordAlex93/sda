Primalidad

En esta sesión vamos a trabajar con números primos y Python.

Criba de Eratostenes

  • En primer lugar, construimos una lista de números hasta donde queremos cribar.
n = 100
numeros = list(range(2, n+1))
  • Procedemos a cribar los números tomando el primero y tachando sus múltiplos, luego el siguiente que no hayamos tachado y así hasta concluir la lista.
def cribado_eratostenes(n):
    """
    Implementación del Cribado de Eratóstenes
    """
    total=0
    # Creamos una lista con todos los números del 2 al n
    numeros = list(range(2, n+1))
    # Iniciamos el cribado
    for i in numeros:
        # Si el número ya ha sido marcado como no primo, saltamos a la siguiente iteración
        if i is None:
            continue
        # Si el número es primo, marcamos todos sus múltiplos como no primos
        total+=1
        for j in range(i*2, n+1, i):
            numeros[j-2] = None
    # Devolvemos una lista con los números primos encontrados
    return total,[num for num in numeros if num is not None]
# Encontramos los primeros números primos
total,primos = cribado_eratostenes(20);
# Imprimimos los resultados
print(f"Total de números encontrados {total}, números primos {primos}")
Atención

Pero este no es el método óptimo ya que realizamos demasiadas iteraciones, hay otro mejor todavía:

# Criba de Eratóstenes optimizada para generar primos hasta n
def criba_eratostenes(n):
    if n < 2: return []
    primos = [True] * (n + 1) # Inicializamos una lista de tamaño n+1. 
    primos[0] = primos[1] = False
    for p in range(2, int(n**0.5) + 1):
        if primos[p]:
            for i in range(p * p, n + 1, p):
                primos[i] = False
    return [i for i, primo in enumerate(primos) if primo]
# Ejemplo de uso
print(criba_eratostenes(100)) # Genera primos hasta 100
Ejercicio

Investiga por qué no es necesario cribar hasta el número \(n\) y nos basta con quedarnos en \(\lfloor\sqrt n \rfloor\).

Solución

Esto es así porque de ser un número compuesto siempre hay, al menos, un factor que es menor o igual a la raiz cuadrada. Si encontramos un divisor ya sabemos que es compuesto.

Comprobar si un número es primo.

Si una lista es “relativamente pequeña”, podemos utilizar un método inmediato de listas:

n = 1000

lprimos = criba_eratostenes(n)
def es_primo1(p):
    if p > n:
        return "Crea una lista más grande"
    if p in lprimos:
        return "Verdadero"
    else:
        return "Falso"

Aquí en este ejemplo empezamos a ver problemas con el tiempo de ejecución y se comprueba que no son buenas funciones, por ello, recurrimos a la función isprime() de la librería sympy:

Si queremos, podemos definir nuestra propia función personalizando la función predefinida:

def es_primo(p):
    if sp.isprime(p):

    else:
        return f'El número {p} no es primo.'

La utilidad que requerimos de los números primos, en esta situación, es para descomponer números en factores primos.

División y resto

Para saber si un número es factor de otro, recordemos que la definición es si la división es exacta, dicho de otra forma; si el resto de la división nos sale \(0\).

Hay una operación para calcular el resto en Python, es el operador %. Y si nos queremos quedar con el cociente, utilizamos //.

def fact1(n):

Pregunta

A la vista de este resultado, ¿qué no estamos teniendo en cuenta?

Por ello, nos conviene utilizar otro algoritmo de factorización mejor.

def factorizar(n):
    factores = []

    for divisor in range(2, int(n**0.5) + 1):
        while n % divisor == 0:
            factores.append(divisor)
            n //= divisor
    
    if n > 1:
        factores.append(n)
    return factores
factorizar(765)

Aplicamos ahora un ejemplo con estilo para factorizar un número:

numero = 765
factores = factorizar(numero)
print(f"{numero} = {'*'.join(map(str, factores))}")
Atención

No obstante, nos conviene utilizar funciones predefinidas para evitar errores.

Vamos a utilizar la función factorint(n) de la librería scipy.

sp.factorint(348)

Normalmente no estamos acostumbrados a ver las factorizaciones de esa forma, con lo que si recurrimos al atributo visual=true, obtenemos una salida como esperamos la factorización de un número.

sp.factorint(348, visual=True)

Máximo Común Divisor

El máximo común divisor nos da el número menor común que divide a los números insertados.

Para calcular el mcd con python, basta llamar a la función gcd(a,b) de la librería scipy:

sp.gcd(4,31)
sp.gcd(20,30)
