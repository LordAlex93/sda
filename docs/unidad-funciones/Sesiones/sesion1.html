Concepto de función

Concepto de función

Una relación \(f\) entre elementos de dos conjuntos \(X\) e \(Y\) se puede representar mediante un diagrama como el que se muestra a continuación:

Ejemplos

1. La relación \(f\) es función de \(X\) en \(Y\), ya que asocia a cada elemento de \(X\) un único elemento de \(Y\).

2. La relación \(g\) no es función de \(X\) en \(Y\), ya que hay elementos de \(X\) que no se relacionan con ningún elemento de \(Y\).

3. La relación \(h\) no es función de \(X\) en \(Y\), ya que hay elementos de \(X\) que tienen más de una imagen.

Definición

Una función \(f\) es una relación entre dos cantidades variables que asocia a cada elemento de un conjunto \(X\) un único elemento de un conjunto \(Y\).

Definición

Si \(x\) es un elemento de \(X\) relacionado con un elemento \(y\) de \(Y\) bajo la función \(f\), se escribe

\[ y=f(x). \]

Como en la expresión \(y=f(x)\), el valor de \(y\) depende del valor de \(x\), se dice que \(y\) está “en función de \(x\)”, y se denomina:

  • Variable independiente a la variable \(x\).
  • Variable dependiente a la variable \(y\).
Ejemplo

  • La función real que relaciona cada número con su cuadrado menos tres unidades se puede representar por:

    \[ f(x)=x^2-3. \]

  • La función real que relaciona cada número con su doble más una unidad se puede representar por:

    \[ g(x)=2x+1. \]

Evaluación de funciones

Definición

Evaluar una función \(y=f(x)\) es obtener el valor que la función le asocia a un valor determinado de \(x\).

Ejemplo

Sea \(f\) una función definida por \(f(x)=5x+4\).

Calcular el valor de la función para \(x=2\) significa hallar \(f(2)\). Para ello, sustituimos \(x\) por \(2\):

\[ f(2)=5\cdot 2+4=10+4=14. \]

Por tanto, la imagen de \(2\) por \(f\) es \(14\).

Gráfica de una función

Definición

La gráfica de una función se obtiene representando en el plano cartesiano todos los pares ordenados \((x,y)\) que cumplen la relación dada por la función \(f\).

En el eje horizontal, llamado eje de abscisas, se colocan los valores de \(x\); en el eje vertical, llamado eje de ordenadas, se colocan los valores de \(y\). Por ello, normalmente trabajaremos con los conceptos “Eje X” y “Eje Y”.

Una gráfica representa una función si ninguna recta vertical la corta en más de un punto.

::: {.callout-important title="Definición"}
Una **función** $f$ es una relación entre dos cantidades variables que asocia a cada elemento de un conjunto $X$ un único elemento de un conjunto $Y$.
:::

::: {.callout-important title="Definición"}
Si $x$ es un elemento de $X$ relacionado con un elemento $y$ de $Y$ bajo la función $f$, se escribe

$$
y=f(x).
$$
:::

Como en la expresión $y=f(x)$, el valor de $y$ depende del valor de $x$, se dice que **$y$** está “en **función** de **$x$**”, y se denomina:

- Variable **independiente** a la variable $x$.
- Variable **dependiente** a la variable $y$.

::: {.callout-note title="Ejemplo"}
- La función real que relaciona cada número con su cuadrado menos tres unidades se puede representar por:

  $$
  f(x)=x^2-3.
  $$

- La función real que relaciona cada número con su doble más una unidad se puede representar por:

  $$
  g(x)=2x+1.
  $$
:::

## Evaluación de funciones

::: {.callout-important title="Definición"}
**Evaluar** una función $y=f(x)$ es obtener el valor que la función le asocia a un valor determinado de $x$.
:::

::: {.callout-note title="Ejemplo"}
Sea $f$ una función definida por $f(x)=5x+4$.

Calcular el valor de la función para $x=2$ significa hallar $f(2)$. Para ello, sustituimos $x$ por $2$:

$$
f(2)=5\cdot 2+4=10+4=14.
$$

Por tanto, la imagen de $2$ por $f$ es $14$.
:::

\Definition[def:grafica-funcion]{
La \textcolor{defblue}{\textbf{gráfica de una función}} se obtiene representando en el plano cartesiano todos los pares ordenados \((x,y)\) que cumplen la relación dada por la función \(f\).

En el eje horizontal, llamado eje de abscisas, se colocan los valores de \(x\); en el eje vertical, llamado eje de ordenadas, se colocan los valores de \(y\). Por ello, normalmente trabajaremos con los conceptos ``Eje X'' y ``Eje Y''.
}
\Observation[vertical]{
\textcolor{observationblue}{\textbf{Prueba de la recta vertical}}

Una gráfica representa una función si ninguna recta vertical la corta en más de un punto.
}
\Example[ej:grafica-cuadratica]{

La siguiente gráfica representa la función cuadrática
\[
f(x)=x^2-3.
\]

\begin{center}
\begin{tikzpicture}
\begin{axis}[
    width=0.72\textwidth,
    height=5.8cm,
    axis lines=middle,
    xmin=-3.5, xmax=3.5,
    ymin=-4.5, ymax=7,
    xtick={-3,-2,-1,1,2,3},
    ytick={-4,-2,2,4,6},
    xlabel={$x$},
    ylabel={$y$},
    grid=both,
    major grid style={line width=.2pt,draw=gray!45},
    minor grid style={line width=.1pt,draw=gray!20},
    samples=150,
    domain=-3:3,
    enlargelimits=false,
    ticklabel style={font=\small},
    label style={font=\small},
]
\addplot[very thick, color=umaBlue] {x^2 - 3};
\end{axis}
\end{tikzpicture}
\end{center}
}

\Question[discusion-funciones]{
\textcolor{questorange}{\textbf{¿Cuáles de estas gráficas representan funciones?}}

Observa las siguientes seis gráficas y decide cuáles representan una función y cuáles no. Justifica tu respuesta utilizando la prueba de la recta vertical.

\begin{center}
\begin{tikzpicture}
\begin{groupplot}[
    group style={
        group size=3 by 2,
        horizontal sep=1.2cm,
        vertical sep=1.4cm
    },
    width=0.27\textwidth,
    height=4.2cm,
    axis lines=middle,
    xmin=-3, xmax=3,
    ymin=-3, ymax=3,
    xtick={-2,2},
    ytick={-2,2},
    grid=both,
    major grid style={line width=.2pt, draw=gray!40},
    minor grid style={line width=.1pt, draw=gray!20},
    ticklabel style={font=\scriptsize},
    title style={font=\bfseries\color{umaBlue}},
    samples=150,
    enlargelimits=false
]

% --------------------------------
% A: Parábola (sí es función)
% --------------------------------
\nextgroupplot[title={A}]
\addplot[very thick, color=umaBlue, domain=-2.2:2.2] {x^2 - 2};

% --------------------------------
% B: Circunferencia (no es función)
% --------------------------------
\nextgroupplot[title={B}]
\addplot[very thick, color=umaBlue, domain=0:360, samples=200]
({2*cos(x)},{2*sin(x)});

% --------------------------------
% C: Valor absoluto (sí es función)
% --------------------------------
\nextgroupplot[title={C}]
\addplot[very thick, color=umaBlue, domain=-3:3] {abs(x)-1};

% --------------------------------
% D: Parábola horizontal (no es función)
% --------------------------------
\nextgroupplot[title={D}]
\addplot[
    very thick,
    color=umaBlue,
    domain=-2.0:2.0,
    samples=200,
    variable=\t
] ({\t^2 - 1},{\t});
% --------------------------------
% E: Función a trozos (sí es función)
% --------------------------------
\nextgroupplot[title={E}]
\addplot[very thick, color=umaBlue, domain=-3:0] {1};
\addplot[very thick, color=umaBlue, domain=0:3] {x-1};

% punto abierto y cerrado para remarcar el cambio
\addplot[only marks, mark=o, mark size=2.5pt, color=umaBlue, fill=white] coordinates {(0,1)};
\addplot[only marks, mark=*, mark size=2pt, color=umaBlue] coordinates {(0,-1)};

% --------------------------------
% F: Recta vertical (no es función)
% --------------------------------
\nextgroupplot[title={F}]
\addplot[very thick, color=umaBlue] coordinates {(1,-3) (1,3)};

\end{groupplot}
\end{tikzpicture}
\end{center}
