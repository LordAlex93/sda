Representación de funciones

Las funciones se pueden describir o representar de diferentes formas. Estas son:

  • Verbal: se nos describe una relación con un enunciado.
  • Tabular: se nos dan los datos en formato tabla.
  • Gráfica: se nos entregan los datos representados con una gráfica.
  • Simbólica: se describe una relación mediante una expresión algebraica.
La letra del DNI como aplicación

El DNI en España es un documento que se entrega a cada persona de nacionalidad española y le identifica de forma única. Dicho documento lleva asociado un número de \(8\) cifras y una letra. Esta letra no es al azar: se considera un dígito de control y nos permite corregir hasta un error cometido en alguna cifra del DNI.

La forma de asociar el número al dígito de control es la siguiente:

  1. Se calcula el resto de dividir el número de tu DNI (sin letra) entre \(23\).
  2. Se asigna la letra al resultado.

La siguiente tabla nos ayuda a asociar el resto con el dígito de control:

T R W A G M Y F P D X B N J Z S Q V H L C K E
